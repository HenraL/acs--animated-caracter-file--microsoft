MeinFreund Download
Hier können Sie das Programm "MeinFreund" herunterladen, das Ihnen die Unterstützung Ihres Freundes bereitstellt. Klicken Sie zum Herunterladen einfach auf die Diskette. Nach dem Download brauchen Sie die Datei einfach nur auszuführen, um die Installation zu starten.

		
	
MeinFreund (komplett)


	
kostenlose 30-Tage-Testversion ohne jegliche Funktionseinschränkung, nach oder während der 30 Tage Testzeit kann das Programm hier günstig bestellt werden.
MeinFreund (ca. 7,0 MB; Version 1.3)

(Enthalten in diesem kompletten Setup: Visual Basic 6 Runtime, MS Agent Control, MS Speech API 4, L&H; Sprachausgabe, deutsche Sprachausgabe, Figur Genie)

		
Systemvorrausetzungen:
- Windows 95 (mit Internet Explorer 4 oder Höher) oder Windows 98/ME/NT4 oder 2000
Installation:
- Ausführen der Datei MeinFreund.exe nach dem Download


Alternative:
Haben Sie bereits den Microsoft Agent mit einem anderen Produkt installiert, so können Sie hier Schritt für Schritt die benötigten Komponenten des Programms herunterladen. So können Sie genau die Downloads sparen, die Sie bereits gemacht haben. Sind Sie sich nicht sicher, welche Komponenten auf Ihrem Computer installiert sind, so installieren Sie einfach das oben angebotene Komplettpaket, es enthält alle für das Programm benötigten Komponenten und das Programm selbst.

Alle diese Komponenten werden von den jeweiligen Herstellern (wie angegeben) kostenlos zur Verfügung gestellt. Bitte beachten Sie, das je nach Betriebssystem einige Schritte hier übersprungen werden können.
